## Installation

Add the dependency to your shard.yml:

```yaml
dependencies:
  read_until:
    github: kojix2/read_until
```

Run `shards install`

## Usage

```crystal
require "read_until"
```

Run example:

```sh
crystal run examples/basic.cr
```

Expected output shape:

```text
endpoint=<host:port>
running=true
latest_read_id=<read-id>
queued_actions=<n>
```
